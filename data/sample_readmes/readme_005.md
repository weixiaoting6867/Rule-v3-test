# Django

Django is a high-level Python web framework that encourages rapid development and clean,
pragmatic design. Django depends on sqlparse and asgiref.

## Python Compatibility

Django supports Python 3.10, 3.11, and 3.12.

## Supported Databases

Django supports PostgreSQL, MariaDB, MySQL, Oracle, and SQLite databases. It integrates
with psycopg2 for PostgreSQL connectivity.

## Features

Django provides an object-relational mapper (ORM). It provides automatic admin interface
generation. Django integrates with Celery for asynchronous task queuing. It also integrates
with Redis for caching and session management.

## Installation

```bash
pip install Django
```

Django is built with Python and supports internationalization and localization out of the box.
