# FastAPI

FastAPI is a modern, fast web framework for building APIs with Python 3.8+ based on standard
Python type hints. FastAPI is built with Starlette and Pydantic.

## Requirements

FastAPI depends on Starlette for the web parts and Pydantic for the data parts.
It supports Python 3.8 and above.

## Features

FastAPI provides automatic interactive API documentation. It integrates with OpenAPI and
JSON Schema. FastAPI supports async and await syntax natively.

## Installation

```bash
pip install fastapi
pip install "uvicorn[standard]"
```

FastAPI integrates with SQLAlchemy, Tortoise ORM, and other async ORMs. It also integrates
with OAuth2 for security and authentication.
