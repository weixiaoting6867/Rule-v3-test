# Flask

Flask is a lightweight WSGI web application framework written in Python. It depends on Werkzeug
and Jinja2. Flask supports Python 3.8 and newer.

## Installation

Install Flask using pip:

```bash
pip install Flask
```

Flask integrates with SQLAlchemy for database access. It also integrates with Celery for
background task processing.

## Features

Flask provides a built-in development server and a fast debugger. It supports RESTful request
dispatching and provides secure cookie support (client-side sessions).
