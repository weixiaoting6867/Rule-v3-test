# Pandas

Pandas is a fast, powerful, flexible and easy to use open-source data analysis and manipulation
tool. It is built with NumPy and depends on python-dateutil.

## Supported Python Versions

Pandas supports Python 3.9, 3.10, 3.11, and 3.12.

## Optional Dependencies

Pandas integrates with Matplotlib for visualization. It also integrates with SQLAlchemy for
reading and writing to SQL databases. Pandas supports reading from Excel files and depends on
openpyxl for .xlsx format support.

## Installation

```bash
pip install pandas
```

Pandas provides DataFrame and Series data structures. It provides powerful, flexible groupby
functionality and supports merging and joining datasets.
